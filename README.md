# Task App Backend 

# Necesary components to run application

- Visual Studio 2022
- .Net Core 6.0 Sdk
- Sql Server 2019 Espress Edition
- Sql Server Management Studio
- Optional (postman)

# Steps to run local application

- Download and install visual studio 2022.
- Download and install .Net Core 6.0 Sdk (link https://dotnet.microsoft.com/en-us/download/dotnet/6.0).
- Download and install Sql Server 2019 Express Edition (link https://www.microsoft.com/es-es/sql-server/sql-server-downloads).
- Download and install Sql Server Management Studio.
- Download code from this repository with commands (git clone https://github.com/dafgonzar/TaskApp.git) if you use tools how Git Bash.
- If you don´t use tools to download code, you can connect the repository with visual studio and download it.
- After you install Sql Server Express Edition you can configurated a user and password to connect the server, this user and password should be on file appsettings.json in "SqlConnection".
- Then you need import the data base backup that i shared.
- After download code in visual studio and make the step 7, you click to run button and thats it.
- When the application is running go back to front-end(react app) and you can create, update, delete tasks.
